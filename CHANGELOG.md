# Changelog

## [1.1.5] - 2026-03-07

### Added
- Screen recording functionality - record display output as MP4 files
- Screenshot capture - save individual screenshots of the current display

### Changed
- Updated download link in settings to point to personal repository releases
- Fixed unused variable declarations in menu and recorder components

### Features
- Full NanoKVM-USB functionality with browser and desktop applications
- 4K video capture and transmission via USB 3.0
- Real-time keyboard and mouse input capture and transmission
- Screen recording (MP4)
- Screenshot capture
- Clipboard support
- Multi-language support (10+ languages)
- Cross-platform support (Windows, macOS, Linux)
